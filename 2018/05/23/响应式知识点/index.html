响应式知识点

发表于 2018-05-23 | 分类于 前端__CSS

媒体查询语法

通过@media写在样式表中

通过标签的media属性为样式表指定设备类型

限制只有视口宽度大于800像素的显示器设备上才能加载文件

使用css的@import指令再当前样示表中按条件引入其它样式表，该方法会增加http请求（影响加载速度）

媒体查询检测的特性

1.width:视口宽度
2.height：视口高度
3.device-width:设备屏幕的宽度
4.device-height:设备屏幕的高度
5.orientation：检测设备处于横向还是纵向
6.aspect-ration:基于视口宽度和高度的宽高比，如aspect-ration: 16/9

meta标签适配移动端

响应式设计

1.用百分比数值：目标元素宽度÷上下文元素宽度
2.用em代替px
3.弹性图片：max-width

媒体查询内的选择器优先级情况

在媒体查询大括号内的的选择器，如果符合查询条件后，相同的选择器权重下的代码会覆盖掉原先的代码

通过适配根元素的font-size值来改变rem,达到响应式开发
